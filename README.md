# 🏡 Holiday

Holiday is a backend-focused vacation rental platform built with **Node.js, Express.js, MongoDB, and EJS**. It allows users to discover, create, and manage vacation property listings through a secure and user-friendly interface.

---

## ✨ Features

- 🔐 Secure user authentication using Passport.js
- 👤 User authorization for protected actions
- 🏠 Create, edit, and delete property listings
- ⭐ Add and manage reviews for listings
- ☁️ Image uploads with Cloudinary cloud storage
- 🗄️ MongoDB database integration
- 📄 Server-side rendering with EJS
- 🎨 Responsive and clean user interface
- ⚡ RESTful routing and CRUD operations
- 💬 Flash messages for user feedback

## 🚀 Installation

### Clone the repository

```bash
git clone https://github.com/pranjalxD/Holiday.git
```

### Navigate to the project

```bash
cd Holiday
```

### Install dependencies

```bash
npm install
```

### Create a `.env` file

Add the following environment variables:

```env
CLOUD_NAME=your_cloudinary_name
CLOUD_API_KEY=your_api_key
CLOUD_API_SECRET=your_api_secret

DB_URL=your_mongodb_connection_string

SECRET=your_session_secret
```

### Start the server

```bash
npm start
```

or

```bash
node app.js
```
